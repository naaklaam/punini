Instalasi
---------
Langkah umum untuk menyiapkan proyek di mesin lokal:

1. Clone repositori:
```bash
git clone https://github.com/naaklaam/punini.git
cd punini
```

2. Instal dependensi (contoh untuk Node.js):
```bash
npm install
# atau
yarn install
```

(Pastikan sesuaikan perintah instalasi dengan stack proyek — ganti instruksi di atas jika proyek ini menggunakan bahasa atau manajer paket lain.)

Cara menjalankan
----------------
Contoh perintah untuk menjalankan proyek secara lokal:

- Mode pengembangan:
```bash
npm run dev
# atau
yarn dev
```

- Build untuk produksi:
```bash

npm start
```

(Ubah perintah di atas sesuai dengan skrip yang tersedia di package.json atau sistem build proyek.)
